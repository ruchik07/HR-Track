
# HR Track

HR Track is a web-based application that allows you to manage a list of employees using CRUD (Create, Read, Update, Delete) operations. It is built with **Spring Boot** and **Thymeleaf**, providing a simple and efficient way to manage employee data.

## Features

- Add new employees
- View the list of all employees
- Update employee details
- Delete employees from the list

## Technologies Used

- **Spring Boot**: Backend framework for building the application
- **Thymeleaf**: Template engine for rendering dynamic web pages
- **Spring Data JPA**: For database interactions
- **Maven**: Dependency management and build tool
